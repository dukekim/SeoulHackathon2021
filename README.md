# 일회용 마스크 쓰래기 수집 장치

* 코로나 시대 전세계 마스크 쓰레기 매월 1290억개 발생

* 이로 인한 환경 문제와 재활용 문제를 해결 방안으로 마스크 수집 장치를 제안 

* 마스크 수집 참여도를 높이기 위하여 게이미피케이션을 적용

* 수집된 마스크 화학 처리를 통해서 플라스틱으로 재활용 (프랑스 업체) 

* 국내 청년이  마스크 150개를 녹여 의자로 만든 사례가 존재하는 등 재활용 연구가 활발히 이루어짐
